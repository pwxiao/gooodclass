"好好好课程表" 是一款专为安徽理工大学学生设计的校园助手小程序，提供便捷的课表查询、成绩查询、空教室查询以及考试安排等功能，帮助同学们更好地规划学习生活。

## 功能特色 ✨

- **查课表**：随时随地查看个人课表，不再担心忘记上课时间。
- **查成绩**：方便快捷地查看各科考试成绩。
- **查空教室**：实时查询校园内的空闲教室，便于自习安排。
- **查考试安排**：了解期末考试时间与地点，避免错过重要考试。
